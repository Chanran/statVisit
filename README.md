# hadoop实验

> 实验要求：自行设计一个网站，记录用户浏览网页时的行为数据，如点击该页面的某个链接行为，数据保存到本地日志文件，经flume收集后并处理,或者用linux定时器任务，上传数据到HDFS中。然后通过HQL查询后生成每日统计数据（PV、UV）保存到关系型数据库MySql中，同时在网站中可以浏览该统计数据。本实验的目标是掌握数据从采集、处理、最终呈现的整个流程。
